As a coding boot camp instructor at The Tech Academy, I create this project to use in a series of tutorial videos to teach students how to use Bootstrap.

This series covered order of importance, how to override bootstrap styling with custom styling, the use of bootstrap layouts (containers, columns, and rows), and layout styling.

  • A responsive feature to collapse the navbar on mobile screens after selecting a nav item, while ensuring it does not close prematurely when a submenu is open.

Simple Recipes

I created this project and wrote the code as a coding boot camp instructor at The Tech Academy. This was then turned into a tutorial series of 10 short 8-10-minute videos.

This project intended to give students at the Tech Academy a basic understanding of CSS, what it is, how to use it within HTML files, the importance of different CSS concepts such as order of importance, the difference between selectors (type, class, and ID), and to practice how different values and combinations can change the styling of an HTML element.

Project Highlights

  • Built on a gradient structure
    • Initial files in the repository cover HTML and CSS only.
    • A separate folder was later added to include JavaScript, keeping base files unchanged for proper tutorial scripting.
  • Javascript Features:
    • Takes all recipe buttons on the page and creates an array.
    • Detects the selected item from the array and determines which button was pressed.
    • Displays a pop-up modal with recipe ingredients and instructions.
    • Implements a form validator to capture and display form data in the console.

IP Tracker

This IP Tracker was via FrontEnd Mentor. I chose to do this challenge on my own to really challenge my JavaScript and API integrations skills. It takes the input of any IP Address and gives the location, zip and state. This project also covered handling user inputs, and presenting data in a user-friendly interface.

*Due to API call limitations I have removed the API key from this project for security reasons but feel free to look around the code on my GitHub.

Project Highlights

  • Accepts any valid IP address as input for tracking.
  • Geo-API Integration: Fetches real-time location data, including the city, region, country, and coordinates associated with the IP address.
  • Map Display: Visualizes the exact location on an interactive map, enhancing the user experience.
